# TECH: `hyperframes init` — scaffold a project

## What it does

Scaffolds a new hyperframes project with the right file structure, copies media, transcribes audio with Whisper, and installs the AI coding skills. Preferred over creating files by hand because it gets the wiring right (`index.html` structure, `hyperframes.json`, asset directories).

## When to use

At the start of every new composition project. Also when cloning a template or example.

## How it works

```bash
# Interactive wizard
npx hyperframes init my-video

# Pick a shipped example
npx hyperframes init my-video --example warm-grain

# Start with a video file
npx hyperframes init my-video --video clip.mp4

# Start with an audio file
npx hyperframes init my-video --audio track.mp3

# Skip prompts (CI / agent automation)
npx hyperframes init my-video --non-interactive
```

### Registry templates (`--example`)

> **Note:** Only `blank` is bundled in the CLI (`templates/generators.ts` `BUNDLED_TEMPLATES`). All other templates are fetched from the Hyperframes template registry (requires internet access) and **fail without internet access**.

- `blank` — empty composition *(bundled — works offline)*
- `warm-grain` — warm palette + grain overlay *(remote registry)*
- `play-mode` — interactive play / pause UI *(remote registry)*
- `swiss-grid` — Swiss design grid composition *(remote registry)*
- `vignelli` — editorial typography *(remote registry)*
- `decision-tree` — decision-flow animation *(remote registry)*
- `kinetic-type` — kinetic typography *(remote registry)*
- `product-promo` — product-demo layout *(remote registry)*
- `nyt-graph` — data-viz / news-graphic style *(remote registry)*

### Side effects

- Creates `my-video/` directory
- Copies provided media into `my-video/assets/`
- Runs Whisper transcription on provided audio (outputs `transcript.json`)
- Installs the hyperframes + hyperframes-cli + gsap skills into the new project's `skills/` folder

## Minimal example

```bash
# Starting from narration.mp3 in CWD
npx hyperframes init acme-launch --audio narration.mp3 --example blank

# Result:
# acme-launch/
#   index.html
#   hyperframes.json        (project config: name, width, height, fps)
#   meta.json               (project metadata: title, description, tags)
#   assets/
#     narration.mp3
#     transcript.json       (auto-generated by Whisper)
#   compositions/           (sub-composition HTML files go here)
#   renders/                (output .mp4 files go here)
#   skills/                 (hyperframes + hyperframes-cli + gsap skills installed)
```

*Attributed to the hyperframes-cli skill — `SKILLS-TO-INTEGRATE/web-design/hyperframes/skills/hyperframes-cli/SKILL.md`.*

## Gotchas

- Interactive mode asks about video type, dimensions, and duration. In CI, pass `--non-interactive` and supply these via flags.
- `--example` templates overwrite `index.html`. Start from `blank` if you want maximum control.
- Whisper transcription can take 1-3 minutes depending on audio length. If you know you don't need transcription, don't pass `--audio`.
