# Tests

Run the full suite:

```bash
uv run --with pytest==9.0.3 pytest tests/ -v
# or, if the dev extra is synced:
uv run pytest tests/ -v
```

The default run is fast and self-contained — anything needing an external tool
(fcvvdp, dev-browser/Chromium) **self-skips** when the tool is absent, so CI
stays green without them.

## Screenshot-parity verification harness

The batch9 integration proves every visual port against its source by
rendering both and comparing them perceptually. Two scripts implement it:

- **`bin/amw-screenshot-compare.sh`** `<reference.png> <distorted.png> [--threshold JOD] [--out DIR]`
  — wraps `fcvvdp` (CVVDP perceptual metric) and exits 0/1 on a JOD threshold.
- **`bin/amw-verify-parity.sh`** `--id <id> --source <url|path> --mine <url|path> [--viewports WxH,...] [--threshold JOD]`
  — renders both at a **fixed viewport** via dev-browser (Playwright pages),
  compares each pair, and writes a report under
  `reports/batch9-verification/<ts±tz>/<id>/` (source/mine PNGs, `score.json`,
  `verdict.md`). `reports/` is gitignored.

### JOD scale and the strict bars

`fcvvdp` emits a **JOD** (Just-Objectionable-Difference) score:
`10.0` identical · `9.0–10.0` barely visible · `8.0–9.0` slight · `7.0–8.0`
noticeable. The harness gates at **JOD ≥ 9.5** for token-identical style tests
and **JOD ≥ 9.0** for technique reimplementations.

### Building `fcvvdp` (one-time, required for parity tests)

`fcvvdp` (halidecx/fcvvdp, Apache-2.0) is a fast C/Zig CVVDP implementation. It
is a **verification-only** dependency — built into gitignored `libs_dev/`, never
shipped, never committed. Requires Zig 0.16.x (`brew install zig`).

```bash
git clone --depth 1 --branch 0.3.0 https://github.com/halidecx/fcvvdp.git libs_dev/fcvvdp
( cd libs_dev/fcvvdp && zig build --release=fast )
# binary → libs_dev/fcvvdp/zig-out/bin/fcvvdp  (or set AMW_FCVVDP_BIN)
```

`dev-browser` + Chromium come from `/amw-init` (or
`npm install -g dev-browser && dev-browser install`).

### Harness tests 🐌

- **`test_screenshot_compare.py`** — unit-tests the compare wrapper against
  deterministic PNG fixtures (generated by `fixtures/gen_parity_fixtures.py`,
  pure stdlib — nothing binary committed). Needs only `fcvvdp`; fast. Self-skips
  when `fcvvdp` is absent.
- **`test_verify_parity.py`** 🐌 — opt-in end-to-end test that drives real
  Chromium renders + fcvvdp. Slow; needs fcvvdp + dev-browser + Chromium. Runs
  only when `AMW_VERIFY=1`:

  ```bash
  AMW_VERIFY=1 uv run pytest tests/test_verify_parity.py -v
  ```

Both are excluded from the fast CI gate (CI has neither tool / leaves
`AMW_VERIFY` unset).
